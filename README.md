# Janim 动画仓库

## 初始化：
```bash
uv sync
```

## 运行：

安装 janim-toolbox

```bash
uv run janim run .\btree_visualization\main.py -i
```
按下 ctrl + j, ctrl + c

## 导出视频：

```bash
uv run janim write .\btree_visualization\main.py 
```
